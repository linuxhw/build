# Build stuff
A list of HW Probe dependencies to build self-sufficient images (AppImage, Snap, etc.).

* hw-probe-1.4-AI (HW Probe 1.4 with backports)
* acpica-unix
* dmidecode
* edid-decode
* hdparm
* hwinfo
* kmod
* libx86emu
* mesa-demos
* pciutils
* usbutils
* smartmontools
* libwww-perl
* perl-HTTP-Date
* perl-HTTP-Message
* perl-LWP-MediaTypes
* perl-Net-HTTP
* perl-parent
* perl-TimeDate
* perl-Try-Tiny
* perl-URI
