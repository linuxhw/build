# Build stuff
A list of HW Probe dependencies to build self-sufficient images (AppImage, Snap, etc.).

* hw-probe-1.4-AI (HW Probe 1.4 with backports)
* hwinfo
* libx86emu
* smartmontools
* hdparm
* edid-decode
* acpica-unix
* kmod
* libwww-perl
* perl-HTTP-Message
* perl-Net-HTTP
* perl-Try-Tiny
* perl-URI
